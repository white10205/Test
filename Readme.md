### 项目介绍

本项目基于uniapp + Vue3搭建的一款壁纸小程序，设配微信小程序、抖音小程序等多端小程序，该小程序由首页、分类页和个人中心页三大页面组成，具有**壁纸搜索、壁纸预览、壁纸下载、分享好友、分享好友、联系客服**等功能。使用unicloud的URL化开发的接口，目前在微信小程序已经上线（微信搜索**咸虾米壁纸**）

### 项目技术栈

**1.Vue3**

**2.Uniapp**

**3.Ajax**

**4.unicloud**
